# WebsiteDetails.md  
# 1. OBJECTIVE

Build a **high-conversion landing website** for a personalized gifting business focused on:
- Fast delivery (12 hours / 24 hours)
- Affordable pricing (starting ₹99)
- Emotional + last-minute gifting use cases

### Primary Goal:
Convert visitors → WhatsApp/DM orders

### Secondary Goals:
- Build trust
- Showcase product quality
- Increase Average Order Value (AOV)

---

# 2. TARGET AUDIENCE

- Age: 16–30
- Location: Ghaziabad, Gaur City, Noida
- Segments:
  - Students
  - Couples
  - Last-minute gift buyers
- Behavior:
  - Mobile-first users
  - Low attention span
  - Decision within 30–60 seconds

---

# 3. CORE VALUE PROPOSITION

**“Personalized gifts in hours, starting at ₹99.”**

---

# 4. WEBSITE STRUCTURE

## Pages:
1. Home (Landing Page) → PRIMARY
2. Designs Gallery (Optional but recommended)
3. Bulk Orders (Optional – later phase)

---

# 5. USER FLOW (CRITICAL)


User lands on website
↓
Sees offer (₹99 + 12-hour delivery)
↓
Scrolls → sees examples + trust
↓
Clicks CTA (WhatsApp)
↓
Conversation starts
↓
Order conversion


---

# 6. DESIGN PHILOSOPHY

### Style:
- Clean
- Warm
- Minimal clutter
- Emotional but not childish

### Colors:
- Primary: Coral / Peach (#FF7F6E)
- Secondary: Teal (#3EC1C9)
- Background: Soft beige (#F7F3EF)
- Text: Dark gray (#333)

### Typography:
- Heading: Rounded, friendly sans-serif
- Body: Clean sans-serif (high readability)

### Tone:
- Friendly
- Fast
- Clear
- Action-driven
